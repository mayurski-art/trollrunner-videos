# MANGO EMPIRE: The Troll Behind It All

Format guide:
- `[MM:SS]` — timecode from start of that part
- **VO** — voiceover line with delivery note
- **OST** — on-screen text (caption burned into video)
- **SFX** — sound effect cue
- **MUS** — music cue or change
